Lung Disease Classifier

This repository contains the full lung disease diagnosis project: a PyTorch CNN model, a Gradio web app, and a simple prediction workflow for chest X-ray screening.

Main project features:

  • Chest X-ray image upload
  • Pneumonia screening result
  • Basic care guidance
  • Local run with python app.py
